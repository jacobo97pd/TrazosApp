Politica de Privacidad de Trazos

Ultima actualizacion: 7 de junio de 2026

Esta Politica de Privacidad explica como Trazos recopila, utiliza, almacena y protege la informacion de los usuarios que utilizan nuestra aplicacion movil.

1. Responsable de la aplicacion

El responsable de la aplicacion es Trazos.

Correo electronico de contacto: soporte.trazos.app@gmail.com

2. Datos que recopilamos

Para ofrecer correctamente las funcionalidades de Trazos, la aplicacion puede recopilar los siguientes datos:

  • Datos de cuenta, como correo electronico, nombre de usuario e identificador de usuario.
  • Datos de ubicacion del dispositivo, necesarios para registrar rutas deportivas y calcular zonas capturadas.
  • Datos de actividad deportiva, como distancia recorrida, trayectos, zonas conquistadas, tiempos y estadisticas de uso.
  • Datos tecnicos del dispositivo, como sistema operativo, identificadores tecnicos, tokens de notificaciones push y registros de errores.
  • Datos relacionados con clubes, rankings, progresos y participacion dentro de la aplicacion.
  • Datos de servicios externos conectados voluntariamente por el usuario, como Strava, si dicha integracion esta disponible.
  • Datos relacionados con compras, suscripciones o funciones premium, si la aplicacion las incorpora.

3. Uso de los datos

Utilizamos los datos recopilados para las siguientes finalidades:

  • Permitir el registro, inicio de sesion y gestion de la cuenta del usuario.
  • Registrar rutas deportivas mediante GPS.
  • Calcular la captura, defensa o perdida de zonas dentro del mapa.
  • Mostrar estadisticas, rankings, clubes y progresos del usuario.
  • Enviar notificaciones relacionadas con actividad, zonas capturadas, zonas robadas, expiraciones o avisos importantes.
  • Prevenir usos fraudulentos, trampas o manipulaciones de ubicacion.
  • Mejorar el rendimiento, estabilidad y seguridad de la aplicacion.
  • Gestionar funciones premium, compras o suscripciones cuando esten disponibles.

4. Uso de la ubicacion

Trazos utiliza la ubicacion del dispositivo para registrar recorridos deportivos y determinar si el usuario ha capturado zonas del mapa.

La ubicacion puede utilizarse mientras el usuario realiza una actividad dentro de la aplicacion. Si la aplicacion permite registrar una carrera en segundo plano, la ubicacion podra seguir utilizandose durante esa actividad aunque la aplicacion no este abierta en primer plano.

El usuario puede desactivar los permisos de ubicacion desde los ajustes de su dispositivo. Si desactiva estos permisos, algunas funciones principales de Trazos podrian dejar de funcionar correctamente.

5. Notificaciones push

Trazos puede utilizar notificaciones push para informar al usuario sobre eventos relevantes, como zonas capturadas, zonas perdidas, expiraciones, actividad de clubes, avisos de seguridad o informacion relacionada con el funcionamiento de la aplicacion.

6. Servicios de terceros

Para prestar sus servicios, Trazos puede utilizar proveedores externos, entre ellos:

  • Firebase Authentication, para el inicio de sesion y autenticacion de usuarios.
  • Cloud Firestore, para almacenar datos de usuario, zonas, rankings, clubes y actividad de la aplicacion.
  • Firebase Cloud Messaging, para el envio de notificaciones push.
  • Firebase Cloud Functions, para ejecutar logica de servidor, validaciones y procesos automaticos.
  • Firebase Storage, si se almacenan archivos o imagenes asociados a la cuenta del usuario.
  • Google Maps Platform, para mostrar mapas, rutas y zonas geograficas dentro de la aplicacion.
  • Strava, unicamente si el usuario decide conectar voluntariamente su cuenta.
  • Servicios de analisis, diagnostico de errores o compras integradas, si estan habilitados en la aplicacion.

Estos proveedores pueden tratar datos conforme a sus propias politicas de privacidad y condiciones de uso.

7. Comparticion de datos

No vendemos los datos personales de los usuarios.

Podemos compartir datos unicamente cuando sea necesario para prestar el servicio, cumplir obligaciones legales, proteger la seguridad de la aplicacion, prevenir fraude o utilizar proveedores tecnicos necesarios para el funcionamiento de Trazos.

8. Conservacion de datos

Conservaremos los datos personales mientras la cuenta del usuario permanezca activa o mientras sean necesarios para prestar el servicio.

El usuario puede solicitar la eliminacion de su cuenta y de sus datos personales escribiendo a soporte.trazos.app@gmail.com.

9. Seguridad

Aplicamos medidas tecnicas y organizativas razonables para proteger los datos del usuario frente a accesos no autorizados, perdida, alteracion, uso indebido o divulgacion no autorizada.

Aun asi, ningun sistema informatico es completamente seguro, por lo que no podemos garantizar una seguridad absoluta.

10. Derechos del usuario

El usuario puede solicitar el acceso, rectificacion, eliminacion, limitacion u oposicion al tratamiento de sus datos personales escribiendo a soporte.trazos.app@gmail.com.

Tambien puede solicitar informacion adicional sobre el tratamiento de sus datos o retirar determinados permisos desde los ajustes de su dispositivo.

11. Menores de edad

Trazos no esta dirigida a menores de 14 anos ni recopila conscientemente datos personales de menores sin el consentimiento valido de sus padres, tutores o representantes legales.

12. Cambios en esta Politica de Privacidad

Podemos actualizar esta Politica de Privacidad cuando sea necesario para reflejar cambios legales, tecnicos o funcionales de la aplicacion.

13. Contacto

Para cualquier consulta sobre esta Politica de Privacidad, puedes contactar con nosotros en soporte.trazos.app@gmail.com.
