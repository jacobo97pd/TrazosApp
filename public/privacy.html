Última actualización: 7 de junio de 2026

Esta Política de Privacidad explica cómo Trazos recopila, utiliza, almacena y protege la información de los usuarios que utilizan nuestra aplicación móvil.

2. Datos que recopilamos

Para ofrecer correctamente las funcionalidades de Trazos, la aplicación puede recopilar los siguientes datos:

  • Datos de cuenta, como correo electrónico, nombre de usuario e identificador de usuario.
  • Datos de ubicación del dispositivo, necesarios para registrar rutas deportivas y calcular zonas capturadas.
  • Datos de actividad deportiva, como distancia recorrida, trayectos, zonas conquistadas, tiempos y estadísticas de uso.
  • Datos técnicos del dispositivo, como sistema operativo, identificadores técnicos, tokens de notificaciones push y registros de errores.
  • Datos relacionados con clubes, rankings, progresos y participación dentro de la aplicación.
  • Datos de servicios externos conectados voluntariamente por el usuario, como Strava, si dicha integración está disponible.
  • Datos relacionados con compras, suscripciones o funciones premium, si la aplicación las incorpora.

3. Uso de los datos

Utilizamos los datos recopilados para las siguientes finalidades:

  • Permitir el registro, inicio de sesión y gestión de la cuenta del usuario.
  • Registrar rutas deportivas mediante GPS.
  • Calcular la captura, defensa o pérdida de zonas dentro del mapa.
  • Mostrar estadísticas, rankings, clubes y progresos del usuario.
  • Enviar notificaciones relacionadas con actividad, zonas capturadas, zonas robadas, expiraciones o avisos importantes.
  • Prevenir usos fraudulentos, trampas o manipulaciones de ubicación.
  • Mejorar el rendimiento, estabilidad y seguridad de la aplicación.
  • Gestionar funciones premium, compras o suscripciones cuando estén disponibles.

4. Uso de la ubicación

Trazos utiliza la ubicación del dispositivo para registrar recorridos deportivos y determinar si el usuario ha capturado zonas del mapa.

La ubicación puede utilizarse mientras el usuario realiza una actividad dentro de la aplicación. Si la aplicación permite registrar una carrera en segundo plano, la ubicación podrá seguir utilizándose durante esa actividad aunque la aplicación no esté abierta en primer plano.

El usuario puede desactivar los permisos de ubicación desde los ajustes de su dispositivo. No obstante, si desactiva estos permisos, algunas funciones principales de Trazos podrían dejar de funcionar correctamente.

5. Notificaciones push

Trazos puede utilizar notificaciones push para informar al usuario sobre eventos relevantes, como zonas capturadas, zonas perdidas, expiraciones, actividad de clubes, avisos de seguridad o información relacionada con el funcionamiento de la aplicación.

El usuario puede desactivar las notificaciones desde los ajustes del dispositivo.

6. Servicios de terceros

Para prestar sus servicios, Trazos puede utilizar proveedores externos, entre ellos:

  • Firebase Authentication, para el inicio de sesión y autenticación de usuarios.
  • Cloud Firestore, para almacenar datos de usuario, zonas, rankings, clubes y actividad de la aplicación.
  • Firebase Cloud Messaging, para el envío de notificaciones push.
  • Firebase Cloud Functions, para ejecutar lógica de servidor, validaciones y procesos automáticos.
  • Firebase Storage, si se almacenan archivos o imágenes asociados a la cuenta del usuario.
  • Google Maps Platform, para mostrar mapas, rutas y zonas geográficas dentro de la aplicación.
  • Strava, únicamente si el usuario decide conectar voluntariamente su cuenta.
  • Servicios de análisis, diagnóstico de errores o compras integradas, si están habilitados en la aplicación.

Estos proveedores pueden tratar datos conforme a sus propias políticas de privacidad y condiciones de uso.

7. Compartición de datos

No vendemos los datos personales de los usuarios.

Podemos compartir datos únicamente cuando sea necesario para prestar el servicio, cumplir obligaciones legales, proteger la seguridad de la aplicación, prevenir fraude o utilizar proveedores técnicos necesarios para el funcionamiento de Trazos.

8. Conservación de datos

Conservaremos los datos personales mientras la cuenta del usuario permanezca activa o mientras sean necesarios para prestar el servicio.

El usuario puede solicitar la eliminación de su cuenta y de sus datos personales escribiendo a [EMAIL DE CONTACTO].

9. Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger los datos del usuario frente a accesos no autorizados, pérdida, alteración, uso indebido o divulgación no autorizada.

Aun así, ningún sistema informático es completamente seguro, por lo que no podemos garantizar una seguridad absoluta.

10. Derechos del usuario

El usuario puede solicitar el acceso, rectificación, eliminación, limitación u oposición al tratamiento de sus datos personales escribiendo a [EMAIL DE CONTACTO].

También puede solicitar información adicional sobre el tratamiento de sus datos o retirar determinados permisos desde los ajustes de su dispositivo.

11. Menores de edad

Trazos no está dirigida a menores de 14 años ni recopila conscientemente datos personales de menores sin el consentimiento válido de sus padres, tutores o representantes legales.

Si detectamos que se han recopilado datos de un menor sin autorización válida, tomaremos medidas para eliminar dicha información.

12. Cambios en esta Política de Privacidad

Podemos actualizar esta Política de Privacidad cuando sea necesario para reflejar cambios legales, técnicos o funcionales de la aplicación.

La versión actualizada estará disponible en esta misma página.
